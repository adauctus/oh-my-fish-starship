## Install

Not yet, but one day...
```fish
$ omf install starship
```

For now, just clone this to ~/.local/share/omf/themes/starship then run

```fish
$ omf theme starship
```

If you've cloned with the existing name, make sure you run instead
```fish
$ omf theme oh-my-fish-starship
```
